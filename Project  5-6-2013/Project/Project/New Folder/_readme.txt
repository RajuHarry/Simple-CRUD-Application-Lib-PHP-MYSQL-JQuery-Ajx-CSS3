AJAX DataGrid

An AJAX enabled datagrid with:

 - pagewise scrolling
 - column sorting
 - in-place editing of data cells with textbox, textarea, select, and checkbox 
   controls.

Documentation:
example.php - documents on defining the column types (textbox,textarea, etc.)
example_mysql.php - demonstrates control linked to actual database.
